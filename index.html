0.1 Introduction

Welcome to the repository for the course Design of Experimental Studies and Applied Statistics in Nutrigenomics — part of the Master’s in Nutrigenomics program.

This repository contains all teaching materials, examples, datasets, and R scripts used throughout the course.
The course is fully reproducible and designed to be explored as a Quarto Book published via GitHub Pages.

0.1.1 📘 Course Overview

This course introduces the statistical foundations of experimental design and applied data analysis in the context of nutrigenomics.
Students will learn to manage, visualize, and analyze omics, clinical, and dietary data using R and reproducible workflows.

0.1.2 Topics

  1. Introduction to Nutrigenomics — study design and data types
  2. Measures of Association
  3. Data Reading and Cleaning in R
  4. Multivariate Descriptive Statistics
  5. Dimensionality Reduction and Clustering
  6. Introduction to Systematic Reviews

Each topic includes lectures, practical labs, and problem sets.

0.1.3 Evaluation Criteria

The evaluation of the course Design of Experimental Studies and Applied Statistics in Nutrigenomics will be based on three main activities, each contributing to the final grade as described below.

Evaluation activity Weight Recoverable Validation mark Improvement of qualification
Expositive seminar 30% No – No
Practices (R exercises) 25% No – No
Final exam 45% Yes 4 No

0.2 Description of the Evaluation Components

0.2.1 1. Expositive Seminar (30%)

  • Format: Group presentation in pairs or groups of three.
  • Objective: To present and discuss a specific topic related to experimental design, statistical modeling, or data analysis applied to nutrigenomics.
  • Evaluation criteria: Clarity, scientific accuracy, statistical interpretation, and ability to engage in discussion.
  • Date: December 4, 2025 (final class session).
  • Deliverables:
    • 15-minute oral presentation.
    • Accompanying Quarto or R Markdown document with code and figures (submitted through the GitHub repository).

0.2.2 2. Practices (25%)

  • Format: Individual R exercises completed during the practical sessions (marked below as Practices).
  • Objective: To apply the statistical techniques discussed in class using real or synthetic omics, clinical, and dietary datasets.
  • Submission: Each student must upload the corresponding .R or .qmd file to their personal GitHub fork or send it through the course platform.

0.2.3 3. Final Exam (45%)

  • Format: Individual written and computational test.
  • Objective: To assess conceptual understanding and practical skills in the design and analysis of studies in nutrigenomics.
  • Date: January 21, 2026
  • Recovery Exam: February 13, 2026
  • Minimum mark required for validation: 4 (out of 10).
